# tsc-ai

AI-driven evasion engine ‚Äî fingerprints your environment, detects sandboxes/AV/EDR, recommends optimal evasion techniques, and adapts beacon timing via reinforcement learning.

## Installation

```bash
npm install -g tsc-ai
```

## Quick Start

### Full environment scan

```bash
tsc-ai scan
```

### Scan + evasion recommendation

```bash
tsc-ai evade
```

### Sandbox/VM detection

```bash
tsc-ai sandbox-test
```

### Adaptive beacon (simulated)

```bash
tsc-ai beacon --interval auto
```

### Train the RL model

```bash
tsc-ai learn --train
```

### Full JSON report

```bash
tsc-ai report --json
```

## Beacon Q-Learning

The adaptive beacon engine uses Q-learning:
- **States**: environment type (normal/sandbox/AV-detected) + network type (NAT/Direct/Proxy) + hour
- **Actions**: beacon intervals (1s, 5s, 30s, 5min, 30min, 1hr) + jitter (0-100%)
- **Reward**: successful beacon ‚Üí higher interval + lower jitter
- **Penalty**: failed beacon ‚Üí lower interval + higher jitter + channel switch
